# Discord Bot Project

This project is a simple Discord bot that can read messages from a specified channel.

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd discord-bot
   ```

2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```
